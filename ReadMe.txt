Data Locker allows you to store any data in encrypted form. A safe haven for your secret data, like a virtual Swiss Bank Locker.

Run main.py to start Data Locker

Data Locker uses python's Crypto module for encryption. So, you may need to install it using $pip install pycrypto

AES encryption algorithm is used for data, and SHA256 is used for password.

Multiple user accounts are supported.
